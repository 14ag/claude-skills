**Contig64.exe** [ADMIN]
Purpose: Defragment individual files or analyse fragmentation.
Flags: `-accepteula -a <path>` (analyse only) | `-s` (recurse subdirectories)
Output: Plain text fragmentation report.
Example: `.\systeminternals\Contig64.exe -accepteula -a C:\Windows\System32\ntoskrnl.exe`
