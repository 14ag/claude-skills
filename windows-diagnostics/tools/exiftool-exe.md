**exiftool.exe** (NirSoft)
Purpose: Read and write metadata from image, audio, video, and document files.
Flags: `<file>` (read all tags) | `-csv <file>` (CSV output) | `-T` (tab-delimited) | `-<tag>=<value> <file>` (write)
Output: Plain text key-value pairs or CSV.
Example: `.\nirsoft\exiftool.exe -csv .\image.jpg`
