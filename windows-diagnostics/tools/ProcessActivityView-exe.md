**ProcessActivityView.exe** (NirSoft)
Purpose: Show file and registry activity of a running process.
Flags: `/stext .\pav_out.txt` to export as tab-delimited text.
Output: Tab-delimited text when exported.
Chaining: Use alongside `Procmon64` for corroboration.
Example: `.\nirsoft\ProcessActivityView.exe /stext .\pav_out.txt`
